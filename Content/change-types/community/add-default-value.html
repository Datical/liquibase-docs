addDefaultValue

The addDefaultValue adds a default value to the database definition for the specified column.

Uses

You can typically use the addDefaultValue when you want to set the default value for the column definition.

Running the addDefaultValue

To add a default value to the database definition for a column, follow these steps:

Step 1: Add the addDefaultValue to your with the needed s as it is shown in the examples. When using the addDefaultValue , you must set one of the following s: defaultValue, defaultValueNumeric, defaultValueBoolean, or defaultValueDate.

Step 2: Deploy your by running the update command.

liquibase update

Now, you should see the default value created.

Available s

Name Description Required for Supports
catalogName The name of the catalog. all
columnDataType The current data type of the column for which to add a default value. informix all
columnName The name of the column for which to add a default value. all all
defaultValue The default value. Either this property or one of the other defaultValue* properties are required. all
defaultValueBoolean The that sets the default value for a column of a boolean type. all
defaultValueComputed The that sets the default value for a column using a function which returns value of the same type as the column has. The is not supported by MySQL.
defaultValueConstraintName The that sets a unique name for default constraint used for a specific column. It works only along with any of defaultValue* s listed. The is supported only by MSSQL.
defaultValueDate The that sets the default value for a column of a date type. For example: timestamp, datetime, date, and others. all
defaultValueNumeric The that sets the default value for a column of a numeric type. For example: integer, bigint, bigdecimal, and others. all
defaultValueSequenceNext The that sets value for a specified column by using the value of the existing sequence. With every new input, the next value of the sequence will be taken. The is not supported by MySQL, MariaDB, SQLite, Ingress, and Sybase.
schemaName The name of the schema. all
tableName The name of the table containing the column. all all
XML example 
<changeSet  author="liquibase-docs"  id="addDefaultValue-example">  
    <addDefaultValue  catalogName="cat"  
            columnDataType="varchar(50)"  
            columnName="fileName"  
            defaultValue="Something Else"  
            defaultValueBoolean="true"  
            defaultValueComputed="now"  
            defaultValueConstraintName="A String"  
            defaultValueDate="2008-02-12T12:34:03"  
            defaultValueNumeric="439.2"  
            defaultValueSequenceNext="seq_name"  
            schemaName="public"  
            tableName="file"/>  
</changeSet>
